Draft Scout tries to evaluate the data (all matches so far) saved by scouting and produce a "ranking" of teams for Draft Alliance selection using the following calculations.   You can vary/change the calculation by changing the modifiers for each item by using the 'Settings' feature in the Options Menu ( ⁝ ) in the Tool Bar at the top right.   These multipliers are represented by UPPER case letters in the formulas below.

Cargo

Scored & Collected
( A*(autoCargoLower+teleCargoLower) + B*(autoCargoUpper+teleCargoUpper)) ✚ (C*(autoCargoFloor+teleCargoFloor) + D*(autoCargoTerminal+teleCargoTerminal)) / # Matches
  • A - Multiplier
  • autoCargoLower - number of Cargo scored in Lower Hub during Autonomous
  • teleCargoLower - number of Cargo scored in Lower Hub during TeleOps
  • B - B>Multiplier
  • autoCargoUpper - number of Cargo scored in Upper Hub during Autonomous
  • teleCargoUpper - number of Cargo scored in Upper Hub during TeleOps
  • C - Multiplier
  • autoCargoFloor - number of Cargo retrieved from Floor during Autonomous
  • autoCargoFloor - number of Cargo retrieved from Floor during TeleOps
  • D - Multiplier
  • autoCargoTerminal - number of Cargo retrieved from Terminal during Autonomous
  • teleCargoTerminal - number of Cargo retrieved from Terminal during TeleOps

Climb

(F*(climbs) ✚ (G*(Hang0) + H*(Hang1) + I*(Hang2) + J*(Hang3) + K*(Hang4)) / # Matches
  • F - Multiplier
  • climbs - number of successful climbs by this robot/team
  • G - Multiplier
  • Hang0 - None
  • H - Multiplier
  • Hang1 - Low
  • I - Multiplier
  • Hang2 - Mid
  • J - Multiplier
  • Hang3 - High
  • K - Multiplier
  • Hang4 - Traversal

Combined

( W*(climbScore) ✚ X*(CargoScore) ✚ Y*(#Penalties) ✚ Z*(#LostComms) ) / # Matches
  • W - Multiplier
  • climbScore - total from the Climb score
  • X - Multiplier
  • CargoScore - total from the Cargo score
  • Y - Multiplier
  • #Penalties - total number of Penalties
  • Z - Multiplier
  • #LostComms - total number of times they Lost Communication
